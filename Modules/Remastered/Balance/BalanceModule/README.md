# Remastered Balance Module

Controls the initial Remastered rates:

- Experience Rate: 10x
- Skill Rate: 3x
- Loot Rate: 2x

The module is controlled by `enable_remastered_balance`.

It does not alter spells, runes, cooldowns, prey, store XP boost, forge,
bestiary, imbuements, vocations, monsters, quests, combat, or database schema.
