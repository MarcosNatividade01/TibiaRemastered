# Ideias

- Launcher com verificacao automatica de atualizacoes.
- Manifest com hash, tamanho e caminho de cada arquivo.
- Canal de atualizacao `dev`, `test` e `stable`.
- Ferramenta para gerar `manifest.json` automaticamente.
- Backup local antes de aplicar atualizacoes.
- Verificacao de integridade do cliente.
- Tela de status do servidor no launcher.
- Documentacao de setup local para Windows.
